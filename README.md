# Driver95 MVP

Простой тренажёр Code 95.

## Что внутри

- 298 вопросов из PDF
- правильные ответы по галочкам
- обучение
- экзамен 40 случайных вопросов
- повтор ошибок
- прогресс в LocalStorage
- без сервера и регистрации

## Запуск
