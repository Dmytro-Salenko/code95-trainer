# Project Backlog (BACKLOG.md)

В данном файле собран бэклог задач, идей и технического долга проекта **Driver95**, распределенный по приоритетам.

---

## 🚨 Critical
* **Анализ метрик релиза v0.3.0 (Июль-Август 2026)**: Двухнедельный сбор статистики (GA4, YouTube, TikTok) и сравнение показателей активности до и после рефакторинга Learn. Код приложения заморожен, новые функции не добавляются.
* **Синхронизация версий кэша**: Регулярный ручной инкремент версии кэша в `service-worker.js` при изменении файлов `app.js`, `index.html` и `style.css`.
* **Тестирование DOM ID перед коммитами**: Исключение падений из-за удаленных/измененных элементов, на которые завязаны обработчики в `app.js`.
