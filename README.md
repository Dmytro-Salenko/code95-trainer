# Driver95 MVP

Простой тренажёр Code 95.

## Что внутри

- выбор базы вопросов на главной: Deutsch / Русский
- немецкая база Austria: 298 вопросов
- русская база StarTruck: 245 вопросов
- обучение
- экзамен 40 случайных вопросов
- повтор ошибок
- прогресс отдельно для каждого языка в LocalStorage
- без сервера и регистрации

## Запуск

Открыть `index.html` в браузере или залить папку на GitHub Pages / Vercel / Cloudflare Pages.
